LOGSTASH, METTEZ DE LA COULEUR DANS VOS LOGS

Ce n’est un secret pour personne, les applications web sont aujourd’hui incontournables. Leur succès et l’augmentation de leur utilisation imposent une surveillance constante des données produites par ce type d’applications. Seulement voilà, il est facile de se retrouver avec une quantité impressionnante de logs. Que faire de ces informations ? Comment les traiter, les stocker, les analyser, etc. ? Faisons le tour de Logstash qui nous propose une réponse simple, efficace, complète et entièrement paramétrable.
